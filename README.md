## Splash Screen App
Aplicativo desenvolvido em Flutter como atividade prática, com o objetivo de criar uma Splash Screen animada e realizar a navegação entre telas com envio de dados.

---

## Funcionalidades
-Splash Screen com animação de entrada (imagem descendo)
-Entrada de dados do usuário (nome e idade)
-Botão para navegação entre telas
-Exibição dos dados na tela Home

---

## Tecnologias Utilizadas
-Flutter
-Dart
-VS Code / Android Studio

---

## Como testar
-Clone o repositório
-Execute flutter pub get
-Rode com flutter run
-Escolha um emulador ou navegador
